# 宋氏集团官网

欢迎访问宋氏集团官网，这里为您提供高质量的图纸设计服务，致力于为新一代的设计带来创新和灵感。

## 网站功能

- **首页**：展示公司简介和核心服务。
- **下载**：提供相关设计工具和资源的下载。
- **设计**：展示我们的设计案例和设计理念。
- **关于**：了解公司背景、团队和联系方式。

## 技术栈

本项目基于以下技术开发：

- **Vue 3**：前端框架，用于构建用户界面。
- **Vite**：快速构建工具，提升开发效率。
- **Element Plus**：UI 组件库，提供丰富的界面组件。

## 开发环境

确保您的开发环境满足以下要求：

- Node.js (版本 >= 14)
- npm 或 yarn
